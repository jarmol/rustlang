      0.1 Description

      This version calculates:

      • the noon time,
      • the daylength,
      • the Sun rise and Sun set times
      • the maximum Sun elevation at the noon time as corrected with the refraction for the given latitude, longitude and time zone on a given date and local time. DLST option is not used here

      If Rust is installed, the program can be run in the shell window with the line command: cargo run
